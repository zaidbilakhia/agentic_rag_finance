## Comparison Against V0, V1, V2, and V3

V0: Simple risk-aware RAG baseline that retrieves and answers directly.

V1: Adds a Query Planner Agent for structured retrieval.

V2: Adds an Evidence Grader Agent for relevance-based filtering before answer generation.

V3: Adds an Answer Critic Agent for self-checking and safer final answers.

V4: Adds a Report Generator Agent that exports the reviewed answer and pipeline metadata into a reusable professional Markdown report.

## Limitations of V4

- V4 does not retrieve, grade, or critique again; it only formats existing pipeline outputs.
- Report quality depends on the final answer and metadata produced by V1-V3.
- The full evaluation run could not be completed in this environment without `OPENAI_API_KEY`.
- The report is Markdown-only and not yet a PDF, DOCX, or dashboard export.
